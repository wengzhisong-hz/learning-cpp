# 线程管理

## 启动线程

线程在`std::thread`对象创建的时候启动。我们可以将一个函数作为参数传入`std::thread`的构造函数：

```c++
void do_some_thing();
std::thread my_thread(do_some_thing);
```

或者传入一个lambda表达式：

```c++
void do_some_thing();
void do_some_thing_else();

std::thread my_thread([]{
  do_some_thing();
  do_some_thing_else();
})
```
